# AI-tools-learning-notes

This Repo is a personal knowledge repository documenting hands-on learning of AI tools, platforms, and ecosystems.

Instead of letting knowledge stay locked in a video or a course, I convert every learning session into structured, reusable notes — so I can:
      
      Revisit concepts quickly without re-watching videos.
      
      Showcase what I've learned to anyone who visits this profile.
      
      Build on previous knowledge as I go deeper into each tool.


If you're learning AI tools too, feel free to use these notes as a reference.
